## 생성 이유

- 공식문서가 아닌 블로그는 틀린 내용이 많아 올바른 자료를 찾아 정리하기 위해 작성

- 공부 중 모르는 키워드가 나올 때마다 비슷한 내용을 반복 검색하며 시간을 낭비하는 것을 줄이고, 한눈에 볼 수 있는 자료를 만들기 위해 작성

- 블로그에 작성하는 것이 번거로워 GitHub 레포지토리의 Markdown 파일로 정보를 정리
